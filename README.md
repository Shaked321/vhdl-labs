# VHDL — Labs

VHDL coursework: digital logic designs simulated in ModelSim and placed &
routed in Intel Quartus.

## Contents

- `lab1/` — AND gate, half adder, mux, counter, 8-bit register,
  synchronous derivative, 7-segment converter, two-flop synchronizer
- `lab2/` — Light-organ design (Quartus project)
- `lab3/` — Stopwatch (Quartus project)
